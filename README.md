# QuizParty-MVP

Веб-приложение для проведения квизов в режиме реального времени.

Проект разработан в рамках учебной практики в формате MVP (Minimum Viable Product).

---

## Возможности

### Организатор

- Регистрация и авторизация
- Создание квизов
- Добавление вопросов
- Запуск комнаты
- Просмотр результатов

### Участник

- Регистрация и авторизация
- Подключение по коду комнаты
- Прохождение квиза
- Просмотр итогового результата
- История участия

## Статус проекта

Проект представляет собой MVP-прототип.

Реализованы:

- интерфейс приложения;
- маршрутизация страниц;
- серверная часть на Express;
- структура REST API;
- модель базы данных.

Полная интеграция с базой данных и WebSocket не выполнялась, так как не входит в минимально жизнеспособную версию продукта.
